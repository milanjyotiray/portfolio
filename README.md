# Milanjyoti Ray | Developer, AI Enthusiast & Security Tech

Welcome to the source code of my personal portfolio website, engineered with pure Vanilla HTML, CSS, and JavaScript.

## Features

- **Semantic Vanilla Architecture:** Zero framework bloat. Built strictly using semantic HTML5, native CSS variables, and modern ES6 JavaScript.
- **Dynamic Typewriter & Particles:** Features a responsive HTML5 Canvas particle background (`particles.js`) and a custom JavaScript Typewriter effect for dynamic Hero subtitle sequencing.
- **Embedded Audio System:** Smooth, toggleable background music integration featuring classical raga that dynamically adapts to browser autoplay restrictions.
- **Contact Integration:** Seamless form submission system routed directly to Google Sheets via Google Apps Script and AJAX fetch logic.
- **Modern Minimalist UI:** Styled explicitly with a Vercel-inspired dark theme, advanced CSS backdrop-filters (glassmorphism), flexbox layouts, and custom keyframe animations.
- **Fully Responsive:** Adapts beautifully across mobile, tablet, and desktop viewports, complete with a custom hamburger navigation menu.

## Technologies Used

- **HTML5:** Core layout and document structure.
- **CSS3:** Flexbox, Grid, Custom Variables, Keyframes, and Media Queries.
- **JavaScript (ES6+):** Intersection Observers (scroll animations), Typewriter logic, Canvas API rendering, and AJAX (`fetch`) for form submission.
- **Lucide Icons:** Used for ultra-clean, lightweight scalable vector icons.
- **Google Fonts:** Utilizing 'Inter' for body readability and 'Outfit' for premium heading typography.

## Local Deployment

To run this project locally, simply serve the directory using any local development server. 

For example, using Python 3:
```bash
python3 -m http.server 5173
```
Then navigate to `http://localhost:5173/` in your browser.
